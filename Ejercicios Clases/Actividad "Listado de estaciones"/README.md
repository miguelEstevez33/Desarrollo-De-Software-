1.1. Listado de Estaciones
📝 Ejercicio 1 — “Listado de Estaciones (HTML puro)”
🎯 Objetivo:

Construir una página HTML que muestre una tabla con el listado de estaciones del sistema de alquiler de bicicletas.
🧩 Requisitos:

    Usar estructura básica HTML5.
    Incluir un encabezado con título de la página.
    Crear una tabla con las siguientes columnas:
        Nro
        Nombre
        Dirección
        Barrio
        Activa
    Completar al menos 4 filas con datos ficticios representando estaciones.

💡 Sugerencias:

    No aplicar estilos CSS (eso se hará en la siguiente clase).
    Cargar el HTML desde VS Code con Live Server.
    Guardar el archivo como estaciones.html.
