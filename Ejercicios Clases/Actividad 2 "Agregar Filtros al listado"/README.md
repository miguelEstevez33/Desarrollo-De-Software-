🧩 Requisitos:

    Agregar un formulario por encima de la tabla, que incluya:
        Campo de texto para buscar por nombre o dirección.
        Select desplegable para elegir un barrio.
        Checkbox para incluir o excluir estaciones inactivas.
        Botón de envío del formulario.
📌 Aclaraciones:

    El formulario no tiene que estar funcional (no necesita buscar realmente, ni usar JS).
    Es importante que se usen correctamente las etiquetas <form>, <label>, <input>, <select>, y <button>.
    El foco está en estructurar correctamente los controles de entrada.
    Para finalizar podemos probar agregarle el atributo method para ver cómo se forma la url
